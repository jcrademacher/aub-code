# Usage

Download and extract the data into the root of this repo. Rename the folder to 'data'. 

Run `phase_decoder_r3.m` to generate uplink doppler results. Run `pie_pulse_error_rate.m` to generate downlink wakeup success, expected retransmissions, and false positive wakeup rates. 

# Version Information

This code requires MATLAB R2024b and above. This code was tested on an 2021 MacBook Pro (M1 Max) on Tahoe 26.5
